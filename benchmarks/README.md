# Benchmarks (Phase 3 — the centerpiece)

The headline artifact of the project. Run the same workload across every tier and fill the
table below from the JSON dropped in `results/raw/` by `smolvla_edge.bench` and the gRPC
client.

## How to produce the rows

```bash
# Tier 1 — Titan X local
python -m smolvla_edge.bench --policy-path <ckpt> --device cuda --precision fp16 \
    --chunking on --tag titanx-fp16 --out benchmarks/results/raw/titanx_fp16.json

# Tier 2 — Xavier NX on-device (sweep precision x chunking)
python -m smolvla_edge.bench --policy-path <ckpt> --device cuda --precision fp16 \
    --chunking off --tag nx-fp16-nochunk --out benchmarks/results/raw/nx_fp16_nochunk.json
python -m smolvla_edge.bench --policy-path <ckpt> --device cuda --precision fp16 \
    --chunking on  --tag nx-fp16-chunk   --out benchmarks/results/raw/nx_fp16_chunk.json
# INT8 goes through a TRT/quantized engine (see deploy/ondevice/), not a naive cast.

# Tier 3 — NX client / workstation server
python deploy/client_server/client.py --server <ws-ip>:50051 \
    --out benchmarks/results/raw/client_server.json

# Collate raw JSON -> results/summary.csv + a markdown table
python benchmarks/collate.py
```
